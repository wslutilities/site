#wslusc

## Manpage Name

wslusc - Component of Windows 10 Linux Subsystem Utility

## Synopsis

`wslusc (-g|-h|-v) ..NAME...`

## Description

This is a WSL shortcut creator to create shortcut on Windows Desktop

## Commands

**-h, --help**
- print a simple help

**-v, --version**
- print current version

**-g, --gui**
- create a shortcut that links to a GUI application
